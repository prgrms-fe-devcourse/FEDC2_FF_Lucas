## 🧞Quick Start 

### 1. Clone & Install Packages

```bash

git clone https://github.com/prgrms-fe-devcourse/FEDC2_FF_Lucas

npm install

```

### 2. Run develop client

```bash
npm start
```
Then, you can access to http://localhost:3000
